# 📊 HR Attrition & Workforce Analytics Dashboard

An interactive **Power BI dashboard** that analyzes employee attrition and workforce trends using HR data. The dashboard provides meaningful insights into employee turnover, department performance, salary distribution, and workforce demographics to support data-driven HR decisions.


## 🚀 Project Objective

The goal of this project is to:
- Analyze employee attrition trends.
- Identify departments and job roles with high attrition.
- Explore workforce demographics.
- Present insights through an interactive dashboard.
- Support better HR decision-making.


## 🛠️ Tools & Technologies
- Power BI
- Power Query
- DAX
- Microsoft Excel



## 📊 Dashboard Features
- Executive KPI Cards
- Department-wise Attrition Analysis
- Salary Band Analysis
- Gender-wise Attrition
- Age Group Distribution
- Education Field Analysis
- Interactive Filters
- Business Insights & Recommendations



## 📈 Key Insights
- Overall Attrition Rate: **16.3%**
- Sales department has the highest attrition.
- Laboratory Technicians record the highest employee exits.
- Employees earning **0–3 LPA** show the highest attrition.
- Employees aged **26–35 years** form the largest workforce group.


## 💡 Business Recommendations
- Improve retention strategies in the Sales department.
- Review compensation for lower salary bands.
- Enhance employee engagement and career development.
- Monitor high-risk job roles using HR analytics.



## 📚 Dataset
**Source:** Kaggle – HR Analytics Dataset
The dataset was cleaned and transformed using **Power Query** before building the dashboard.
